# Free's Tech Blog - Source Code

这个仓库包含了 [Free's Tech Blog](http://blog.556756.xyz) 的所有源代码。

## 简介

本博客基于 [Hexo](https://hexo.io/) 框架构建，使用了 [Fluid](https://github.com/fluid-dev/hexo-theme-fluid) 主题。

所有内容通过 Markdown 格式撰写，并利用 Cloudflare Pages 实现自动化部署（详细看[Cloudflare Pages 部署 hexo 博客](https://blog.556756.xyz/2026/02/15/hexo-cloudflare%E5%8D%9A%E5%AE%A2%E6%90%AD%E5%BB%BA/) ）。
